# xuandiao1

全国各省份**定向选调生**招录信息汇总，涵盖 2025 届与 2026 届招考公告、官方入口链接及招录说明。

## 资料目录

- [定向选调汇总.md](./定向选调汇总.md) — 全国 31 个省级行政区定向选调生招录公告汇总表，含官方直达链接、招录对象说明、届别状态及招考时间线参考。

## 说明

- **定向选调**：由省委组织部主导，面向全国重点高校（985/211/双一流）应届优秀毕业生，定向分配至基层岗位，晋升通道优先。  
- **区别于普通选调**：本仓库仅收录定向选调相关信息，不含普通选调（非定向）公告。  
- 各省 2026 届公告预计于 2025 年秋季（9–10 月）陆续发布，请持续关注各省委组织部官网。
