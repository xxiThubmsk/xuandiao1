# 全国各省份定向选调生招录公告汇总（2025届 & 2026届）

> **说明**
> - **定向选调**：面向全国重点高校（"985"/"211"或教育部直属高校）应届优秀毕业生，由省委组织部主导，定向到基层艰苦岗位工作，属于省级统一管理的选调生计划。  
> - **普通选调**：由省级人社/公务员主管部门面向全省普通高校毕业生开展，条件相对宽松，**本表不收录**。  
> - 各省公告时间每年略有差异，以官网最新公告为准；带"🔗"的链接为官方直达页面或历年公告索引页。  
> - 2026届公告多在 2025 年下半年或 2026 年春季陆续发布，以下标注"**待发布**"的省份请持续关注官网。

## 四、定向选调 vs 普通选调 区别

| 对比维度 | 定向选调 | 普通选调 |
|----------|----------|----------|
| 招录主体 | 省委组织部（高规格） | 省人社厅 / 省公务员局 |
| 学历要求 | 一般要求 985/211/双一流 应届硕士、博士（部分含本科） | 普通全日制本科及以上 |
| 院校要求 | **严格限定重点高校** | 无特殊院校要求 |
| 岗位去向 | 定向到县乡基层，含艰苦地区 | 各级机关均有 |
| 晋升通道 | 省级统一管理，提拔较快 | 常规公务员晋升路径 |
| 绑定期 | 一般 2–5 年基层服务期 | 视岗位而定 |

## 六、注意事项

1. **以官网为准**：本表链接均指向各省委组织部官网首页或选调生栏目入口，具体公告页面 URL 每年更新，请在官网内搜索"定向选调"获取当年最新公告。
2. **核实招录对象**：务必仔细阅读公告中"招录对象及条件"一节，确认所在学校及专业是否在招录范围内。
3. **区分定向与普通**：各省网站上同时存在"定向选调"和"普通选调（非定向）"公告，招录条件差异较大，报考前须严格区分。
4. **2026届公告时间**：截至本文整理时（2025年），多数省份2026届公告尚未发布，预计2025年秋季（9–10月）陆续推出，届时将更新本表。
5. **数据更新**：欢迎通过 PR 或 Issue 补充最新公告链接和招录数据。
